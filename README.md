# bilibot_Vertx

## 介绍

bilibot是一款还在开发阶段的b站私信聊天机器人框架，使用vertx框架作为底层框架

* QR扫码登录 ✔
* 会话管理,连续对话管理 ✔
* 插件化应用管理 ✔
* 消息发送队列 ✔
* 文本消息发送 ✔
* 图像消息发送 ✔
* 模拟站内分享 ✖(未来版本)
